## ‚ú® Phase 2 Implementation Rules

### For Text Elements
```xaml
‚úÖ CORRECT:
<Label Style="{StaticResource TitleSmallStyle}" Text="Plant Name" />

‚ùå WRONG:
<Label FontSize="16" Text="Plant Name" />
```

### For Colors
```xaml
‚úÖ CORRECT:
<Label TextColor="{AppThemeBinding Light={StaticResource DarkGray}, Dark={StaticResource Gray100}}" />

‚ùå WRONG:
<Label TextColor="#2F2F2F" />
```

### For Cards
```xaml
‚úÖ CORRECT:
<Border Style="{StaticResource ModernCardStyle}" Padding="16">

‚ùå WRONG:
<Border CornerRadius="10" Padding="10">
```

## üìã Common Development Patterns

### Pattern 1: Styled Label
```xaml
<Label 
    Style="{StaticResource BodyStyle}" 
    Text="{Binding PropertyName}" />
```

### Pattern 2: Titled Section
```xaml
<Label Style="{StaticResource TitleStyle}" Text="Section Header" />
<Label Style="{StaticResource BodyStyle}" Text="Section content" />
```

### Pattern 3: Modern Card
```xaml
<Border Style="{StaticResource ModernCardStyle}">
    <StackLayout Spacing="12" Padding="16">
        <Label Style="{StaticResource TitleSmallStyle}" Text="Title" />
        <Label Style="{StaticResource BodyStyle}" Text="Content" />
    </StackLayout>
</Border>
```

### Pattern 4: Form Field
```xaml
<StackLayout Spacing="8">
    <Label Style="{StaticResource LabelStyle}" Text="Field Label" />
    <Entry 
        Style="{StaticResource BodyStyle}"
        Text="{Binding FieldValue}" 
        Placeholder="Enter value" />
</StackLayout>
```

### Pattern 5: Action Button
```xaml
<Button 
    Style="{StaticResource ...}"
    Command="{Binding ActionCommand}"
    Text="Action" 
    CornerRadius="12" 
    HeightRequest="48" />
```

## üêõ Troubleshooting

| Issue | Solution |
|-------|----------|
| Build fails | Check XAML syntax, verify closing tags |
| Colors look wrong | Verify AppThemeBinding, check Colors.xaml |
| Text size inconsistent | Use typography styles, never hardcode |
| Dark mode broken | Ensure AppThemeBinding in all colors |
| Layout issues | Check spacing grid (8px multiples) |
